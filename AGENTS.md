# INVYRA: guía rápida para agentes

INVYRA crea invitaciones digitales premium. Antes de cambiar páginas o demos, consulta:

- [ADN de marca](docs/INVYRA-DNA.md)
- [Reglas de paquetes](docs/PACKAGE-RULES.md)
- [Checklist de demos](docs/DEMO-CHECKLIST.md)
- [Matriz de QA](docs/QA-MATRIX.md)

Reglas mínimas:

- Conserva el enfoque móvil como invitación vertical y el escritorio como composición editorial contenida.
- Usa **Esencial** como nombre oficial del primer nivel. Signature adapta sus módulos al evento sin reducir profundidad, contenido ni valor equivalente.
- No añadas un control visible de música. Intenta reproducirla al abrir la experiencia y páusala cuando la página pierda visibilidad.
- Toda demo comercial debe tener WhatsApp funcional. Cuando el RSVP sea “No asistiré”, deshabilita los campos dependientes y excluye sus datos del envío.
- Memorial es una demo privada conmemorativa, fuera de paquetes, catálogo y conteo comercial. “Privada” significa no listada; GitHub Pages no aporta control de acceso.
- No publiques datos privados, credenciales, endpoints sensibles ni información personal de clientes.
- No inventes alcances de paquetes; marca las dudas como **POR CONFIRMAR**.
- No instales dependencias ni hagas commit, push, merge o pull request sin autorización expresa.
